##About
####HatClient is a custom client for forks of [hack.chat](https://github.com/AndrewBelt/hack.chat "hack.chat")
![Login Screen](http://paswd4.com/s/n5cu3e.png)

###This is an **EARLY DEMO**, some features do not work as intended.
